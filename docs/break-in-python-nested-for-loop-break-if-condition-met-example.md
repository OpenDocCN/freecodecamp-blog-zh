# Python 中的 Break–满足条件时嵌套循环中断示例

> 原文：<https://www.freecodecamp.org/news/break-in-python-nested-for-loop-break-if-condition-met-example/>

编程中的循环让我们连续执行一组指令/代码块，直到满足某个条件。

我们还可以使用循环来迭代一组数据，并对数据集中的每一项执行类似的操作。

```
names = ["John", "Jane", "Doe"]
for i in names:
    print(i)

"""
John
Jane
Doe
"""
```

上面是一个 Python `for`循环，它遍历一个名字列表并打印所有的名字。

在我们想要在到达最后一项之前或者在满足给定条件之前停止迭代的情况下，我们可以使用`break`语句。`break`语句有自己的条件——这告诉它何时“打破”循环。

在本文中，我们将首先看看如何在`for`和`while`循环中使用`break`语句。然后，我们将研究一些可以用来打破 Python 中嵌套循环的方法。

## 如何用 Python 写一个`break`语句？

您在希望终止的循环中定义了`break`语句。在这一节中，我们将看到如何在`for`和`while`循环中使用`break`语句。

### 如何在`for`循环中使用`break`语句

这里有一个例子:

```
names = ["John", "Jane", "Doe"]
for i in names:
    print(i)
    if i == "Jane":
        break
```

在上面的代码中，我们打印了一个姓名列表:

```
for i in names:
    print(i)
```

然后我们创建了一个新的条件，它检查`i`变量何时获得了一个等于“Jane”的名称。当满足该条件时，要求循环停止。它停止是因为当`i`是“简”时`break`语句停止循环:

```
if i == "Jane":
        break
```

这就相当于说:“把所有的名字都打印出来，一到简就停下来”。所以在我们的控制台中，在这三个名字中，只有“John”和“Jane”会被打印出来。

我们也可以用数字来做这件事:

```
for i in range(10):
  print(i)
```

上面的代码打印了从 0 到 9 的一系列数字。但是我们可以阻止这个循环打印所有的数字——我们可以停在一个特定的数字上。方法如下:

```
for i in range(10):
  print(i)
  if i == 5:
      break
```

现在循环在 5 点停止。所以我们在控制台只会看到 0，1，2，3，4，5。

### 如何在`while`循环中使用`break`语句

本节中的示例将类似于上一节中的示例。我们将使用一个`while`循环来代替。

```
i = 0
while i < 10:
  print(i)
  i += 1
```

上面的代码打印了从 0 到 9 的一系列数字。我们将使用`break`在数到 5 时停止打印数字。

```
i = 1
while i < 10:
  print(i)
  if i == 5:
    break
  i += 1
```

就像我们在上一节中所做的一样，我们创建了一个新的条件:`if i == 5`并且当这个条件被满足时，循环被终止，而不是一直打印到 9。

### 如何在嵌套循环中使用`break`语句

在这一节中，我们将看到如何在嵌套循环中使用`break`语句。

下面是嵌套循环的样子:

```
for x in range(4):
    for y in range(4):
        print(x, y)

"""
0 0
0 1
0 2
0 3
1 0
1 1
1 2
1 3
2 0
2 1
2 2
2 3
3 0
3 1
3 2
3 3
"""
```

嵌套循环经常让初学者感到困惑。因此，如果您想知道我们是如何得到上面的输出(在代码中被注释掉)的，这里有一个简短的解释:

会给出一个从 0 到 3 的数字范围。

只打印`for x in range(4):`会给我们 0，1，2，3。但是我们在循环中嵌套了另一个数字范围:`for y in range(4):`

第二个循环所做的是按照它所拥有的整数个数复制每个数字`x`(在`y`范围内)。我们有四个数字在`y`范围内——0，1，2，3。

所以对于`x`范围，第一个数字是 0，会出现四次。每次出现都从`y`范围取一个数；即:0 和 0，0 和 1，0 和 2，0 和 3。这也适用于`x`范围内的其他数字。

如果您仍然觉得难以理解，那么尝试自己运行代码。尝试更改每个循环的范围，看看输出会发生什么。

让我们使用`break`语句来打破这个循环:

```
for x in range(4):
    for y in range(4):
        if x == 1:
            break
    print(x, y)

"""
0 3
1 3
2 0
3 3
"""
```

尽管上例中的循环似乎已经停止，但仔细看看输出(上面的注释)，您会意识到外部循环仍然打印出它的所有值，这不是我们想要的。

所以仅仅使用`break`语句实际上并不会破坏嵌套循环。让我们来看看实现我们想要的结果的一些变通方法。

#### 使用布尔变量

```
force_break_loop = False
for x in range(5):
    for y in range(5):
        if x == 2:
            force_break_loop = True
            break
    if force_break_loop:
        break
    print(x, y)

"""
0 4
1 4
"""
```

在上面的例子中，我们使用了一个布尔变量，它的初始值是`False`。当循环到达预期的断点时，我们将这个布尔值设置为`True`，但这还不是全部。我们检查变量何时为`True`，然后指定`break`语句。

#### 使用 Break 语句两次

```
for x in range(5):
    for y in range(5):
        if x == 3:
            break
    if x == 3:
        break
    print(x, y)

"""
0 4
1 4
2 4
"""
```

在这个例子中，我们定义了两个`if`语句——都返回一个`break`语句来强制循环中断。

虽然这些变通办法可能在给定的实例中终止了循环，但是您会注意到，每次循环中断时，内部循环的值都保持不变。

这并不意味着嵌套循环是不好的。但是在您的逻辑中实现它们之前，请确保您确实需要它们。
