# 打破代码 2.0:一个浏览器游戏，你解决使用编码技能的任务

> 原文：<https://www.freecodecamp.org/news/break-the-code-2-game/>

《破解密码 2.0》是一款新的浏览器游戏，将你带回到 1999 年。

你用你的编程知识和解谜直觉来完成破译密码的任务。

游戏分布在 4 个任务中:

*   /G1/drive-" BoxOfChocolates "
*   /G2/drive—“搏击俱乐部”
*   和/g3/和/g4/驱动器，您可以通过解决前两个驱动器来解锁
