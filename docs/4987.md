# npm 初始化的最佳时间

你应该在什么时候？

大多数开发人员在创建并导航到一个新项目后马上运行`npm init`。

在项目开始时对`npm init`有意义，因为我们使用 npm 来下载依赖项。一旦我们`npm init`，我们就可以开始下载(并保存)我们的依赖项。

对于大多数项目来说，这种工作流程是可行的。
