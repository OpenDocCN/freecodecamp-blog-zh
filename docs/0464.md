# 免费编码训练营-通过 Class Central + freeCodeCamp 学习编码

> 原文：<https://www.freecodecamp.org/news/free-coding-bootcamp-learn-to-code-with-class-central-and-freecodecamp/>

2021 年，班级中心团队帮助我和我的朋友教授了三批基于 freeCodeCamp 课程的免费编程训练营。

来自全球的 20，000 多名学习者加入了我们的行列，能够与这么多的学习者一起学习是一种荣幸和快乐。

我和拉蒙在教学中度过了如此美好的时光，以至于我们想再回来参加另一批培训。**从 9 月 5 日**开始，我们将围绕 freeCodeCamp 的[网络开发](https://www.freecodecamp.org/learn/2022/responsive-web-design/)和 [JavaScript](https://www.freecodecamp.org/learn/javascript-algorithms-and-data-structures/) 课程，教授另一个为期 6 周的免费训练营。

## 训练营是如何运作的？

就像上次训练营一样，我们通过观看课程的直播来支持学习者，并给你一个实时提问的机会。

如果我们没有按照您的时间表播放，请不要担心，所有视频都可以点播。我们也有一些很棒的[客座演讲人](https://www.youtube.com/watch?v=XOIDnNA9N7c)来增加额外的背景，一个社区支持的论坛，和一个活跃的(和适度的)不和谐的社区。

这些课程是为兼职远程学习者设计的，我们的大多数学习者每周花 5-20 个小时在训练营上。虽然有可能同时选修两门课程，但我们(轻轻地！)建议一次只做一件事，这样有助于你在学习时保持舒适的工作量。

我们试图将学习体验设计得轻松愉快。不需要参加考试，不需要点名，如果你花了超过 6 周的时间来完成课程材料，也不会有什么不好的事情发生。

我们将继续改进我们的教学方式和支持学习者的方式。如果你想加入并和我们一起学习，你可以在这里注册[网页开发课程](https://www.classcentral.com/cohorts/webdev-bootcamp-fall-2022)或者在这里注册 [JavaScript 课程](https://www.classcentral.com/cohorts/js-bootcamp-fall-2022)。

## 你会学到什么？

这些课程将涵盖 freeCodeCamp 响应式网页设计和 JavaScript 课程中的所有内容。

这意味着 Web 开发训练营将教授:

*   HTML 基础:语法，元素，图像，链接，列表，表单，按钮。
*   CSS:选择器，类，id，属性，字体，颜色，大小，位置。
*   易访问和响应的网页设计:面向所有人和所有设备的网站。
*   CSS 布局工具:用 CSS Flexbox 和 CSS grid 定位元素。

JavaScript 课程将涵盖:

*   编程基础:语法，变量，操作，循环，函数。
*   数据结构:数组，多维数组，数组方法，对象。
*   算法:搜索、排序、反转、合并、拼接、哈希。
*   面向对象编程:方法，构造函数，原型，继承。
*   函数式编程:术语，约束，映射，过滤。

## 这个课程和付费训练营有什么不同？

我们的免费训练营旨在支持尽可能多的学员，因此我们无法提供一对一的学员支持。这一次，我们做了更多的同伴支持学习，并引入付费导师来帮助更好地支持学习者。由于这些课程的规模，我们也无法提供职业安置支持。

最令人兴奋的不同是，这些课程是免费的。付费训练营的费用可能高达 1 万美元或更多。我们设计了这些课程，这样你就可以把钱省下来用在重要的事情上，比如房租和电子游戏。

## 训练营是如何免费的？

很棒的问题！拉蒙和我志愿贡献我们的时间，因为我们热爱教学。

班级中心团队提供平台并为导师付费，因为他们对技术教育充满热情。

freeCodeCamp 免费提供课程，因为这是他们的使命。

## 怎样才能把课程做得更好？

作为一名教师，我总是很乐意了解更多关于我们如何更好地支持学习者的信息。我特别感兴趣的是，我们如何更好地支持母语不是英语的学习者或有硬件挑战的学习者。

如果您参加了本课程，请告诉我们如何才能更好地为您服务。我们不可能做到每个人都需要的一切，但是听到更多关于你如何学习的信息会对我们所有人都有帮助。
