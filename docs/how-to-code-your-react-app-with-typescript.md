# 如何用 TypeScript 编写 React 应用程序

> 原文：<https://www.freecodecamp.org/news/how-to-code-your-react-app-with-typescript/>

React 是用于构建用户界面的最流行的前端 JavaScript 库之一。对 React 项目使用 TypeScript 可以产生更健壮的代码库，错误更少。

我们刚刚在 freeCodeCamp.org YouTube 频道上发布了一个完整的课程，将教你如何使用 React with TypeScript。

路边编码器开发了这个课程。他是一位受欢迎的讲师，创建了各种框架和主题的编程课程。

在本课程中，您将首先学习 TypeScript 的基础知识。然后，您将学习如何通过构建一个出色的项目来将 TypeScript 集成到 React 应用程序中。

您将学习如何将 TypeScript 与 React 挂钩(如 useState、useRef 和 useReducers)一起使用。您将学习如何通过定义组件的道具类型将道具从一个组件传递到另一个组件。你会学到更多！
