# Python 中的推文可视化和情感分析

> 原文：<https://www.freecodecamp.org/news/tweet-sentiment-analysis-python/>

在这个 Python 教程中学习分析推文。Vincent Russo 展示了如何使用 Tweepy 模块直接从 Twitter 实时传输实时推文。推文被可视化，然后使用 TextBlob 模块对推文进行情感分析。

这是视频的片段:

*   流媒体直播推文
*   光标和分页
*   分析推文数据
*   可视化推文数据
*   情感分析
