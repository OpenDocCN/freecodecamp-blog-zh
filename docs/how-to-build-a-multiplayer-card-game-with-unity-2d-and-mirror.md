# 如何建立一个多人卡牌游戏与统一 2D 和镜子(更新)

> 原文：<https://www.freecodecamp.org/news/how-to-build-a-multiplayer-card-game-with-unity-2d-and-mirror/>

在 Unity 2D 中使用画布起初会感觉复杂，特别是如果你试图在学习 C#脚本的同时学习编辑器的话。

考虑将一个单人游戏转变为多人游戏也可能令人生畏，因为有许多新概念要考虑，还有几个第三方包可供选择。

在这个新视频系列的第 1 部分中，我们将在 Unity 中创建一个基本的 2D 纸牌游戏，使用画布、C#脚本、预设和更多工具(1 小时 10 分钟观看)创建随机卡片组和可拖动/可放下的卡片:

[https://www.youtube.com/embed/videoseries?list=PLCbP9KGntfcFbPnUSuQXLKoaJ8MiWBSk6](https://www.youtube.com/embed/videoseries?list=PLCbP9KGntfcFbPnUSuQXLKoaJ8MiWBSk6)

在第 2 部分中，我们将通过清理嵌套的画布预设并启用鼠标悬停卡片预览(42 分钟观看)来继续冒险:

[https://www.youtube.com/embed/eo6-FIT3G0M?feature=oembed](https://www.youtube.com/embed/eo6-FIT3G0M?feature=oembed)

在第 3 部分中，我们将增加基本的多人游戏功能，包括[镜像](https://mirror-networking.com/) (1 小时 26 分钟的观看时间):

[https://www.youtube.com/embed/X7rKFVDGT64?feature=oembed](https://www.youtube.com/embed/X7rKFVDGT64?feature=oembed)

在第 4 部分中，我们将继续与 Mirror 合作来改善我们的多人游戏体验(36 分钟观看):

[https://www.youtube.com/embed/4gQgfJmZ7C0?feature=oembed](https://www.youtube.com/embed/4gQgfJmZ7C0?feature=oembed)

在第 5 部分中，我们将把我的一个游戏 *[Entromancy:黑客大战](https://www.nightpathpub.com/hacker-battles)* 中的游戏逻辑添加到我们的多人游戏项目中(2 小时 34 分钟观看):
