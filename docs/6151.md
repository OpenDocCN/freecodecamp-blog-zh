# JavaScript 游戏开发简介

> 原文：<https://www.freecodecamp.org/news/intro-to-game-development-with-javascript/>

*   项目设置和画布
*   在画布上画画
*   清理画布
*   创建玩家的桨
*   游戏循环
*   处理键盘输入
*   移动船桨
*   停止划桨
*   绘制球的图像
*   移动球
*   重构游戏类
*   球和桨之间的碰撞检测
*   装载砖块
*   建筑标高
*   砖块碰撞检测
*   暂停屏幕
*   简单的开始菜单
*   游戏结束
*   完成级别
