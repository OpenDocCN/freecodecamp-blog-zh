### 我们如何构建学习小组目录应用程序

我们的学习小组目录应用程序是一个简单的单页静态 web 应用程序，运行在 GitHub 页面上。这样，我们可以很容易地添加新的学习小组，并立即部署对代码的改进。

德克萨斯州圣安东尼奥的教师兼开发人员威廉·麦克唐纳开发并维护了这款应用。

像我们所做的一切一样，它是完全开源的。

如果你碰巧经营一个组织，并且你想要一个这样的目录来存放你所有的本地分会，请随意使用。
