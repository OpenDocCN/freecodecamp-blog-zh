# 如何使用函数式 JavaScript 编写经典游戏 Snake 并在您的浏览器中运行它——包含代码示例的完整教程

> 原文：<https://www.freecodecamp.org/news/how-to-code-the-classic-game-snake-and-play-it-in-your-browser-using-functional-javascript-a-full-tutorial/>

还记得上世纪 90 年代诺基亚每部手机上都预装的游戏《贪吃蛇》吗？你在屏幕上操纵一条蛇，试图在不把它撞到墙上或它自己成长的身体上的情况下喂养它。

这个游戏开发教程将向你展示如何构建经典的贪吃蛇游戏，并让它可以在你的浏览器中运行。

你不会使用任何图书馆，以太。只是标准的 JavaScript。您还将学习一些函数式编程！
