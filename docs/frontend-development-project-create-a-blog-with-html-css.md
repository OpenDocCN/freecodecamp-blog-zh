> 原文：<https://www.freecodecamp.org/news/frontend-development-project-create-a-blog-with-html-css/>

是时候通过创建博客来提高你的 HTML 和 CSS 技能了。

我们刚刚在 freeCodeCamp.org YouTube 频道上发布了一个前端开发课程，将教你如何用 HTML 和 CSS 建立一个博客。
