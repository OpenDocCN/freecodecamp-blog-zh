# 学习反应-初学者的完整课程

> 原文：<https://www.freecodecamp.org/news/learn-react-course/>

在 Bob Ziroll (@bobziroll)的这个完整课程中，学习使用 React JavaScript 库构建动态、交互式 web 应用程序的所有基础知识。

完成本课程后，您将了解在 React 中构建 web 应用程序所需的一切。

本课程涵盖:

*   JSX
*   道具和状态
*   条件渲染
*   样式组件
*   生命周期方法
*   从 API 获取数据
*   处理事件
*   表单和受控组件
*   编写现代 React 代码
*   设置环境
