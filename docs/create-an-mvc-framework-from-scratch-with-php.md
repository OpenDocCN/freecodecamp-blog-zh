# 用 PHP 从头开始构建你自己的 MVC 框架[免费 6 小时课程]

> 原文：<https://www.freecodecamp.org/news/create-an-mvc-framework-from-scratch-with-php/>

web 开发中经常使用诸如 Ruby on Rails、Laravel、Angular 等 MVC 框架。MVC 代表“模型视图控制器”，是分离应用程序主要部分的常用方法。

我们在 freeCodeCamp.org YouTube 频道上发布了一门课程，将教你如何使用 PHP 从头开始构建自己的 MVC 框架。

该课程将让你更好地理解这些框架在幕后是如何工作的。

您在本课程中构建的 MVC 框架并不完全适合生产。但是它会教你如何将这些基本概念应用到你自己的 web 开发项目中。

这门课程由经验丰富的 PHP 开发人员 Zura Sekhniashvili 教授，他在 YouTube 上被称为“代码狂热者”。

您可以在家编写代码，并学习如何实现以下 MVC 框架特性:

*   自定义路由
*   设计者
*   控制器
*   视图/布局
*   模型
*   迁移
*   表单小部件类
*   请求数据的处理
*   确认
*   注册/登录
*   简单活动记录
*   会话简讯
*   中间件
*   应用程序事件
*   框架可重用/可安装核心

当您实施这些功能时，您将对以下内容有更深刻的理解:

*   如何使用文件系统
*   如何使用数据库
*   如何在实际项目中使用抽象
*   如何实现自定义路由
*   如何使用 composer 安装第三方软件包
*   如何创建可重用的 composer 包
*   和其他重要的技能，您可以将它们应用到未来的软件开发项目中。
