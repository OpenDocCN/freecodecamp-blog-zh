# 学习 Gatsby，一个静态站点生成器

> 原文：<https://www.freecodecamp.org/news/learn-gatsby-version-3/>

盖茨比是一个网站生成器。这是一个基于 React 的免费开源框架，可以帮助开发者快速构建网站和应用。

我们刚刚在 freeCodeCamp.org YouTube 频道上发布了一个 9 小时的 Gatsby 课程，将教你如何使用最新版本的框架。
