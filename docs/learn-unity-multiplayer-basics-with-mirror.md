# 通过镜像学习 Unity 多人基础

> 原文：<https://www.freecodecamp.org/news/learn-unity-multiplayer-basics-with-mirror/>

Unity 是游戏开发中最知名和最成熟的引擎之一， [Mirror](https://mirror-networking.com/) 是一个第三方开源网络解决方案，允许你在游戏中加入多人游戏。

在这个新的教程中，我们将带着镜像浏览 Unity 多人游戏开发的基础！
