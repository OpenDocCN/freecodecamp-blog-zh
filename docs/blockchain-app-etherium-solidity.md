# 使用以太坊智能合约和可靠性构建您的第一个区块链应用程序

> 原文：<https://www.freecodecamp.org/news/blockchain-app-etherium-solidity/>

在 Dapp 大学的这个视频课程中，学习如何使用以太坊智能合约构建一个区块链应用程序。智能合约是驻留在以太坊区块链上特定地址的代码和数据的集合。

您将学习如何使用 Solidity 编程语言创建一个带有以太坊智能合约的 todo 应用程序。您还将学习编写测试、部署到区块链，以及创建客户端应用程序。

你可以在[freeCodeCamp.org YouTube 频道](https://www.youtube.com/watch?v=coQ5dg8wM2o)观看完整的视频课程(1.5 小时观看)。
